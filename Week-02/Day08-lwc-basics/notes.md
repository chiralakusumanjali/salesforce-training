Salesforce Summer Program – Day 8 Quick Notes

LWC (Lightning Web Components)

* Modern Salesforce UI framework
* Built using HTML, JavaScript, CSS, and XML
* Supports reusable and modular components
* Provides fast performance and scalable UI development

---

Frontend
Handles:

* User Interface (UI)
* Buttons
* Forms
* User interactions
* Display and navigation

Frontend technologies:

* HTML
* CSS
* JavaScript
* LWC

---

Backend
Handles:

* Database operations
* Business logic
* Security validation
* Record processing
* Data management

Backend technologies:

* Apex
* SOQL
* Salesforce database services

---

Advantages of Component-Based Architecture

* Reusability
* Faster development
* Easier maintenance
* Better scalability
* Consistent UI
* Simplified testing
